Sorter
=========

The Sorter is a library for to understand and release benchmarks.

The project is organized under the same architecture found in [Catalano Framework]. Some sort algorithms implemented:

  - Bubble Sort
  - Bogo Sort
  - Insertion Sort
  - Shell Sort
  - Merge Sort
  - Heap Sort
  - Quick Sort
